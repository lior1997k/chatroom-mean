# ChatRoom (MEAN + Socket.io)

## Folders
- `Backend/` — Node/Express + Socket.io + MongoDB
- `Frontend/` — Angular + Angular Material

## Quick start

### Backend
```bash
cd Backend
npm install
cp .env.example .env
# edit .env with real values
npm start
```


### Backend
```bash
cd Frontend
npm install
npm start
# opens http://localhost:4200
```
